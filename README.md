## 한국어

### 3×3 숫자 퍼즐 게임

브라우저에서 즐길 수 있는 클래식 8-퍼즐 게임입니다. 빈칸을 이용하여 숫자 1부터 8까지 순서대로 배열하는 두뇌 트레이닝 게임입니다.

### 기능 특징

- **다국어 지원**: 한국어, 영어, 중국어, 일본어 지원
- **브라우저 언어 자동 감지**: 사용자의 브라우저 설정에 따라 자동으로 언어 선택
- **반응형 디자인**: 모바일과 데스크톱 모두에서 최적화된 UI
- **퍼즐 풀이 알고리즘**: 항상 풀 수 있는 퍼즐 생성 보장
- **언어 전환 기능**: 실시간 언어 변경 지원

### 게임 방법

1. 3×3 그리드에 숫자 타일이 무작위로 배치됩니다.
2. 빈칸과 인접한 타일을 클릭하여 이동합니다.
3. 숫자를 1부터 8까지 순서대로 배열하면 성공!

### 기술 스택

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript

### 실행 방법

1. 저장소 클론:
   ```bash
   git clone https://github.com/kkalrak/8puzzle.git
   ```
2. 브라우저에서 `index.html` 파일 열기

### 다국어 지원

이 게임은 4가지 언어를 지원합니다:
- 한국어 (기본)
- 영어
- 중국어
- 일본어

언어는 브라우저 설정을 기반으로 자동 감지되며, 우측 상단의 언어 버튼으로 언제든지 변경할 수 있습니다.

## English

### 3×3 Number Puzzle Game

A classic 8-puzzle game playable in your browser. Arrange numbers 1 through 8 in order using the empty space for brain training.

### Features

- **Multilingual Support**: Korean, English, Chinese, Japanese
- **Browser Language Auto-detection**: Automatically selects language based on user's browser settings
- **Responsive Design**: Optimized UI for both mobile and desktop
- **Puzzle Solvability Algorithm**: Guarantees always solvable puzzles
- **Language Switching**: Real-time language change support

### How to Play

1. Number tiles are randomly arranged on a 3×3 grid.
2. Click tiles adjacent to the empty space to move them.
3. Arrange numbers from 1 to 8 in order to win!

### Tech Stack

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript

### How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/kkalrak/8puzzle.git
   ```
2. Open `index.html` in your browser

### Multilingual Support

This game supports 4 languages:
- Korean (default)
- English
- Chinese
- Japanese

Language is auto-detected based on browser settings, and can be changed anytime using the language buttons in the top-right corner.

## 中文

### 3×3 数字拼图游戏

可在浏览器中玩的经典8拼图游戏。使用空白空间将数字1到8按顺序排列进行脑力训练。

### 功能特点

- **多语言支持**: 韩语、英语、中文、日语
- **浏览器语言自动检测**: 根据用户的浏览器设置自动选择语言
- **响应式设计**: 针对移动设备和桌面设备优化的UI
- **拼图可解算法**: 保证始终可解的拼图生成
- **语言切换功能**: 实时语言更改支持

### 游戏方法

1. 数字方块随机排列在3×3网格上。
2. 点击空白空间相邻的方块进行移动。
3. 将数字从1到8按顺序排列即可获胜！

### 技术栈

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript

### 运行方法

1. 克隆存储库：
   ```bash
   git clone https://github.com/kkalrak/8puzzle.git
   ```
2. 在浏览器中打开 `index.html` 文件

### 多语言支持

此游戏支持4种语言：
- 韩语（默认）
- 英语
- 中文
- 日语

语言基于浏览器设置自动检测，并可使用右上角的语言按钮随时更改。

## 日本語

### 3×3 数字パズルゲーム

ブラウザでプレイできるクラシック8パズルゲームです。空きスペースを使って1から8までの数字を順番に並べる脳トレゲームです。

### 機能特徴

- **多言語サポート**: 韓国語、英語、中国語、日本語
- **ブラウザ言語自動検出**: ユーザーのブラウザ設定に基づいて自動的に言語を選択
- **レスポンシブデザイン**: モバイルとデスクトップの両方で最適化されたUI
- **パズル解決可能性アルゴリズム**: 常に解決可能なパズル生成を保証
- **言語切り替え機能**: リアルタイムの言語変更サポート

### 遊び方

1. 3×3グリッドに数字タイルがランダムに配置されます。
2. 空きスペースに隣接するタイルをクリックして移動します。
3. 数字を1から8まで順番に並べると成功！

### 技術スタック

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript

### 実行方法

1. リポジトリをクローン：
   ```bash
   git clone https://github.com/kkalrak/8puzzle.git
   ```
2. ブラウザで `index.html` ファイルを開く

### 多言語サポート

このゲームは4つの言語をサポートしています：
- 韓国語（デフォルト）
- 英語
- 中国語
- 日本語

言語はブラウザ設定に基づいて自動検出され、右上の言語ボタンでいつでも変更できます。
